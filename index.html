About

What is Chinchilla? It's whiteboard. Your friends can join to server which you created, then, you and your friends can draw on one whiteboard and chat on this. It's powered by Ruby and QT so you need it in your OS.

Requirements

  • gem qtbindings (tested on 4.6.3.2)
  • ruby (tested on 1.9.2)

Installation

There is no installation. : ) Simply run ./chinchilla. Maybe you need to `chmod +x chinchilla` before first run. If you search logs, they are in ~/.config/chinchilla/logs (only on server's computer!)
